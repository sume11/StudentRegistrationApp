Java Servlet web application with MySQL database for student registration.
     Features
Student registration form
MySQL database storage
View all students in table
Servlet and JSP
  Setup
Create MySQL database: studentdb
Create table: students(id, name, email, year)
Update DatabaseConnection.java with your MySQL credentials
   Deploy to Tomcat
   Access: http://localhost:8080/StudentRegistrationApp/register.html
Files
src/ - Java servlets, DAO, model, database connection
web/ - HTML form and JSP pages
web.xml - Servlet configuration
